deployment
**********


Description
===========

A container for your OCI GoldenGate resources, such as the OCI
GoldenGate deployment console.


Available Commands
==================

* change-compartment

* collect-diagnostics

* create

* delete

* export-wallet

* get

* import-wallet

* list

* start

* stop

* update

* upgrade

* upgrade-to

* wallet-exists
