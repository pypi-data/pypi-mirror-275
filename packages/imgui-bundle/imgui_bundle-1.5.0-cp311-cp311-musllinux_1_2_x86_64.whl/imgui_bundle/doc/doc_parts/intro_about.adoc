=== Dear ImGui Bundle

include::_utils.adoc[]

// tag::motto[]
https://github.com/pthom/imgui_bundle[Dear ImGui Bundle] is a comprehensive bundle for https://github.com/ocornut/imgui[Dear ImGui], featuring various powerful libraries from its ecosystem. Designed to facilitate the creation of applications in C++ and Python across Windows, macOS, Linux, iOS, Android, and emscripten (Web apps), it is ideal for application developers, and researchers eager to dive into GUI development with ease and efficiency. This bundle simplifies the process, allowing you to focus on the creative aspects of your projects.
